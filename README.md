# expense-sharing-
Overview
A simplified backend implementation of an expense sharing application similar to Splitwise.
The focus is on core business logic such as expense splitting, balance tracking, and settlement.
Features:
Create groups,
Add shared expenses,
Supported split types:
Equal;
Exact;
Percentage,
Track and simplify balances,
View how much a user owes and is owed.
Design:
Backend-only implementation,
Uses in-memory data structures,
Balance simplification to minimize transactions,
UI-independent and easily extensible.
